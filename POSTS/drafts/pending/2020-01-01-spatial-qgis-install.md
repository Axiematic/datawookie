## Add Repository and Update

{% highlight bash %}
$ sudo add-apt-repository ppa:ubuntugis/ppa
$ sudo apt-get update
{% endhighlight %}

## Install

{% highlight bash %}
$ sudo apt install qgis
{% endhighlight %}
